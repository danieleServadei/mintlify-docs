## Invitations

The invitation is a way to invite a new user to your organization.

This is the process to invite a new user:
<Steps>
<Step title="Create a new invitation">
From the invitations page (accessible from organization page), create a new invitation by clicking on the "Create Invitation" button.
</Step>
<Step title="Send the invitation">
After creating the invitation, copy the invitation link and send it to the user you want to invite.
</Step>
<Step title="User registration">
The user, following the invitation link, will be able to create an account and join your organization.
</Step>
</Steps>

From the invitations page, you can list all the invitations associated with your organization, renew an expired invitations and delete the active invitations.

<Warning>Invitations are valid for 7 days. After this period, the invitation will expire and the user will not be able to join the organization using that invitation.</Warning>

<Warning>Invitations are unique and can be used only once. After the user has registered, in the invitations page you can see who has used the invitation.</Warning>
